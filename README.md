>文件使用说明(更新时间：2026/5/29)

>1.完整装配好的装配体名为：瓦力（其他的装配体是为了方便建模）
>
>2.想查看完整装配体的同学可以在下载完文件后将文件里的装配体和零件放到同一个文件中，这样打开主装配体就不会出现零件丢失，因文件太大无法上传，才拆成多个文件
